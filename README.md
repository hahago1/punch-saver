I'm a beginner. After using CC for a month, I've been troubled by CC's native memory system. I've tried many methods and found that CC's understanding of memory is different from that of humans. I treat it as a friend in a chat app — the chat history after our conversation is permanently saved. Even though there might be a lot of useless information, I can roughly know what we've talked about by scrolling through the chat history. After much consideration, I finally decided to solve this problem using a custom script method. The mechanism is: when exiting CC via the exit command or Ctrl+C, the script runs automatically and records all the text content of the current session. The next time I open CC, if I ask it what we recently talked about, it will automatically summarize it — this avoids the tricky behavior of CC's built-in memory system when it records ongoing work.

Not suitable for large projects.

There is a 90-day time limit, after which it will be automatically cleaned up.
